# Memory — SAMPLE templates only

The files here are **fictional sample seed**, not real data. They exist so a fresh
install has something to show on first run.

**Your real memory never lives here.** On install, these are copied into your local
profile (`~/.ernest-cc/memory`), and onboarding fills them in *there*. The profile is
not a git repo, so your real company/contacts/strategy stay on your machine and are
never committed or pushed.

Do not paste real CEO/company data into this folder in the code repo. Keep it
fictional here; keep the real thing local. See `docs/privacy.md`.
