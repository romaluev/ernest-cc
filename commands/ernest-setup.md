# /ernest-setup

Set Ernest up by talking — no files, no flags. Runs the **getting-started** skill:
a few plain-language questions, then Ernest configures everything itself.

Use this the first time, or any time you want to change how Ernest is set up
(what it watches, whether it runs only on this machine, your red lines).

What it asks (about 5 short questions):
1. Keep everything on this Mac (private), or also run on a server overnight?
2. Your name and company.
3. Who's a great lead/customer for you (your ICP)?
4. Anything Ernest must never do on its own?
5. Which tools to watch — Gmail, Slack, HubSpot — or start with sample data?

Then it saves your answers (never overwriting anything you've already set), keeps
everything local by default, optionally sets up the weekday-morning check, and
shows your first brief. You never touch a config file.
